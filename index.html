Computer Networks Web Technology Software Engineering

  • TCP/IP based computer Network is a

    Packet-switched network

  • Network architecture has a stack of layers. Which of the following is not true for this architecture?

    Network protocols cannot work with multiple layer

  • Fill in the blanks: OSI model have ___ layers, and TCP/IP model have ___ layers.

    5,7

  • How many headers and trailers are added by transport layer in TCP/IP protocol suit?

    1 header and no trailer

  • What are the end network devices?

    Servers, smartphones and computers

  • What is the basic difference between LAN and WAN

    LAN works with a small area while WAN covers a large geographic area.
    LAN connection need to be set up by an organisation while WAN connection need to be rented from a service provider

  • Match following properties with the corresponding protocol layers:
    I. It provides end-to-end data delivery system.
    II. It does not provide reliability, but deliver packets to a remote destination.
    III. This layer provides device-to-device communication.
    IV. Sometimes, it is considered as address of a host.
    V. End users uses this layer directly.
    VI. It can use almost any network interface available

    Application layer-> V; Transport layer-> I; Network Layer-> II, IV; Data link layer-> VI, III

  • Who defines the Internet architecture?

    IETF

  • The attribute used to define a new namespace is.

    xmlns

  • Which HTML attribute is used to define inline styles?

    style

  • Which is the correct CSS syntax?

    body {color: black}

  • _________ keyword is used to declare variables in javascript.

    var

  • Which one of the following is not a symptom of the present software crisis:

    Software products are required to perform very complex tasks.

  • Model building is a form of which one of the following:

    Abstraction

  • A software process model represents which one of the following?

    The way in which software is developed.

  • Which one of the following activities spans all stages of a software development life cycle(SDLC)?

    Project management
